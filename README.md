# Blessings Maburuse Portfolio Website

## Project Description
This is a fully responsive personal portfolio website designed to showcase my skills, projects, and contact details as an aspiring web developer.

## Technologies Used
- HTML5
- CSS3

## Features
- Fully responsive design (mobile, tablet, desktop)
- Consistent navigation across all pages
- Semantic HTML5 structure
- CSS Grid layout for projects section
- Contact form with HTML5 validation

## Setup Instructions
1. Download or clone the repository
2. Open index.html in any web browser
